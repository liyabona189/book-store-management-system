# 📚 Book Store Management System

## Overview

A console-based Book Store Management System developed in C++ that allows users to manage books, members, and special members through an interactive menu-driven interface.

## Features

- Add books
- Add members
- Add special members
- View all books
- View member information
- Object-Oriented Programming design
- Inheritance
- Encapsulation
- Constructors
- Menu-driven interface

## Technologies

- C++
- Visual Studio

## Concepts Demonstrated

- Object-Oriented Programming
- Classes and Objects
- Constructors
- Getters and Setters
- Inheritance
- Arrays
- Functions
- User Input Validation

## Future Improvements

- File handling
- Search by ISBN
- Delete books
- Update book details
- Better exception handling
